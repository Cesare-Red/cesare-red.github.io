# Scuderia Iron GitHub Pages

## Deploy
1. Crea un repository GitHub.
2. Carica `index.html`, `css/`, `js/`, `assets/`.
3. Attiva GitHub Pages da `main` / root.
4. Sostituisci `https://formspree.io/f/CHANGE_ME` con il tuo endpoint Formspree.

## Asset
- `assets/cal.jpeg` = logo circolare.
- `assets/caladrino.png` = foto principale.

## Email form
Per cambiare la mail di destinazione, modifica l'endpoint Formspree collegato al form.
